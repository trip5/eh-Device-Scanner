# eh Device Scanner

A cross-platform mobile app for discovering IoT devices on your local network using the **ehDP (eh Discovery Protocol)**.

## Overview

eh Device Scanner is a Flutter-based mobile application that provides fast, zero-configuration discovery of ehDP-compatible devices. The app sends UDP broadcast packets on your local network and displays responding devices in a simple, easy-to-use interface.

This app serves as both a practical tool for users and a reference implementation of the ehDP client specification.

Unfortunately, I know very little about coding for a mobile app so a lot of this was generated by co-pilot.  As a result, I kept its functionality very basic. It requires no permissions.

## Features

- **Fast Discovery**: Discovers devices in 2-3 seconds
- **Zero Configuration**: No setup required, just tap and scan
- **Device Details**: Displays device name, IP address, type, and version
- **Web UI Access**: Opens device web interfaces in your system browser

## ehDP Protocol

This app implements the **ehDP v1** protocol specification. ehDP is a lightweight, UDP-based discovery protocol designed for ESP8266/ESP32 and similar IoT hardware.

It is a discovery protocol implementable in ESPHome and Arduino-based firmware.  Read more about the protocol (and projects that use it) [here](https://github.com/trip5/ehDP).

**Protocol Details:**
- **Port**: 11867 (UDP)
- **Discovery Payload**: `EHDP_DISCOVER`
- **Response Format**: JSON (single UDP packet)
- **Protocol Version**: `ehdp/1`

## Installation

### Android
1. Download APK from [Releases](https://github.com/trip5/eh-Device-Scanner/releases)
2. Enable "Install from Unknown Sources"
3. Install and launch

### iOS

## Usage

1. Connect to WiFi
2. Tap "Scan for Devices"
3. Wait 2-3 seconds
4. Tap device to open web UI

## Requirements

- **Android**: 5.0+ (API 21+)
- **iOS**: 12.0+
- **Network**: Local WiFi only
